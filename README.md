## Getting Started

## Frontend

Clone this repo:

```bash
git clone https://github.com/anisapatel/nc-daily-newsy.git

cd nc-daily-newsy
```

Install the dependencies:

`npm i`

Run the app. It will run on http://localhost:3000.

`npm start`
